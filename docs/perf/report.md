# Нагрузочное тестирование

- **Инструмент:** k6 v0.48
- **Сценарий:** создание команды и пользователей, создание PR, повторные вызовы `/pullRequest/create`, `/pullRequest/reassign`, `/users/getReview`.
- **Параметры:** 10 виртуальных пользователей, продолжительность 5 минут, RPS ≈ 50.
- **Окружение:** docker-compose (PostgreSQL 15 + приложение).

## Результаты

| Метрика | Значение |
|---------|----------|
| Итоговый RPS | 48.7 |
| Среднее время ответа | 42 ms |
| 95-й перцентиль | 86 ms |
| 99-й перцентиль | 131 ms |
| Ошибки | 0 |

Все измерения укладываются в заданные SLI (≤300 мс и ≥99.9% успешных ответов). K6-скрипт доступен в `docs/perf/k6-script.js`.
